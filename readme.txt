# Проект Web-site "Гороскоп"

## 1. Задачи проекта

Создать веб-сайт "Гороскоп".


## 2. Установка

Установить с помощью пипа:

	$ pip install -r requirements.txt


## 2. Установка

Для работы запустить два файла:
	* horoscope.py - веб-приложение
	* delete_and_fill_db.py - скрипт для удаления и заполнения базы данных


## 3. Возможности

1. Домашняя страница:
    url: /home
    Вы можете:
	- выбрать гороскоп для своего знака зодиака, кликнув на его картинке
	- использовать форму, чтобы получить персональный гороскоп
	- нажать на "Новости" и прочесть интересные статьи из мира астрологии
	
2. Новости:
    Реализуются с помощью базы данных и обновляются каждую ночь в 00:00

3. Персональный гороскоп:
    * введите своё имя и дату рождения (пол по желанию)
    * получите персональный гороскоп на сегодня/завтра/неделю/месяц
